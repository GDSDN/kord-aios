# Story Definition of Done (DoD)

Use this checklist before marking a story as complete.

## Mandatory
- [ ] Code compiles without errors
- [ ] All tests pass
- [ ] Linting/type checks pass
- [ ] Story tasks and acceptance criteria are checked off
- [ ] Story status updated to the correct phase
- [ ] PR is ready (description, reviewers, links)

## Optional
- [ ] Documentation updated
- [ ] Changelog entry added
- [ ] Accessibility checks completed (if UI changes)
- [ ] Performance/regression checks completed
